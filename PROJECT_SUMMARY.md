# 프로젝트 요약

## 1. 프로젝트 목표

학생들이 대학교 수강 기록 텍스트 파일을 업로드하면, 해당 내용을 분석하고 정리하여 웹 페이지에 시각적으로 보여주는 것을 목표로 합니다.

## 2. 주요 기능

*   **사용자 관리:** 회원가입 및 로그인 기능
*   **수강 기록 분석:** 텍스트 파일(.txt)을 파싱하여 학기별 수강 과목 정보 추출
*   **결과 시각화:** 분석된 수강 기록을 학기별로 그룹화하여 웹 페이지에 표시
*   **데이터 영속성:** H2 데이터베이스를 사용하여 사용자 정보 및 분석 결과 저장

## 3. 기술 스택

*   **언어:** Java
*   **프레임워크:** Spring Boot
*   **데이터베이스:** H2
*   **프론트엔드:** HTML, Thymeleaf

## 4. 핵심 로직

*   **`CourseService.java`:**
    *   파일 파싱: 정규식을 사용하여 텍스트 파일에서 수강 정보를 추출합니다.
    *   학기 그룹화: 정규 학기와 계절학기를 구분하여 학기 번호를 부여합니다. 특히, 계절학기는 직전 정규 학기에 0.5를 더한 값(예: 3.5학기)으로 처리하여 별도 표시합니다.
    *   데이터베이스 저장: 분석된 결과를 `SemesterCourse` 엔티티로 변환하여 H2 데이터베이스에 저장합니다.
*   **`CourseController.java`:**
    *   사용자 요청 처리: 로그인, 회원가입, 파일 업로드, 결과 조회 등의 HTTP 요청을 처리합니다.
    *   데이터 전달: `CourseService`에서 받은 데이터를 `Model`에 담아 `Thymeleaf` 템플릿으로 전달합니다.

## 5. 데이터베이스 모델

*   **`User`:** 사용자 정보를 저장하는 엔티티 (id, username, password)
*   **`SemesterCourse`:** 학기별 수강 과목 정보를 저장하는 엔티티 (id, semester, courseName, grade, user)
