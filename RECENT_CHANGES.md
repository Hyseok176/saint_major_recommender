# Recent Changes

## 9. 사용자 피드백 기반 기능 개선 (2025-08-16)
*   **추천 제외 기능**:
    *   과목 추천 페이지에서 각 과목 옆에 'X' 버튼을 추가했습니다.
    *   'X' 버튼을 누르면 해당 과목이 추천 목록에서 제외되고, 즉시 다른 과목으로 대체되어 동적인 추천 경험을 제공합니다.
*   **'재수강' 과목 표시**:
    *   이미 수강한 과목을 장바구니에 담을 경우, '나의 수강이력' 페이지의 다음 학기 목록에 '재수강' 라벨을 표시하여 명확하게 구분하도록 했습니다.
*   **UI 편의성 개선**:
    *   **여러 통계 비교**: '전체 과목' 페이지에서 여러 과목의 통계 차트를 동시에 열어두고 비교할 수 있도록 수정했습니다.
    *   **ESC 키 연동**: 장바구니가 열려 있을 때 키보드의 `ESC` 키를 누르면 패널이 닫히도록 하여 사용 편의성을 높였습니다.
*   **'전체 과목' 페이지 UI 수정**:
    *   과목 목록에서 항상 보이던 '담기' 버튼을 숨기고, 사용자가 과목 통계 차트를 펼쳤을 때만 차트 옆에 표시되도록 변경했습니다.

## 8. 추천 시스템 고도화 및 UI/UX 개선 (2025-08-16)
*   **추천 알고리즘 개선**:
    *   단순 수강생 수 기반 추천에서 **학기 근접도**와 **수강생 수**를 모두 고려하는 **가중치 점수 기반 추천**으로 로직을 변경했습니다.
    *   `Score = Σ (1 / (1 + |나의 현재 학기 - 타 수강생의 수강 학기|))` 공식을 도입하여, 사용자와 비슷한 시기에 수강한 학생이 많을수록 높은 점수를 받도록 개선했습니다.
*   **추천 점수 툴팁 기능 추가**:
    *   추천 과목 목록에 **정보 아이콘(ⓘ)**을 추가하여, 마우스를 올리면 추천 점수에 대한 상세 정보를 볼 수 있도록 했습니다.
    *   툴팁에는 최종 점수, 총 수강 인원, 그리고 '평균 학기 근접도'가 표시되어 추천 근거를 투명하게 제공합니다.
*   **수강 이력 페이지 기능 추가**:
    *   **학기별 접기/펼치기**: 각 학기 목록을 접고 펼 수 있는 아코디언 기능을 추가하여 가독성을 높였습니다.
    *   **장바구니 연동**: 장바구니에 담은 과목들이 '다음 학기' 섹션에 자동으로 표시되도록 하여 수강 계획 수립을 돕도록 했습니다.
*   **버그 수정 및 안정성 강화**:
    *   툴팁에서 줄 바꿈(`\n`)이 텍스트로 표시되던 문제를 HTML 렌더링을 활성화하여 해결했습니다.
    *   장바구니에서 항목을 삭제하거나 패널 외부를 클릭해도 패널이 닫히지 않도록 하여 사용성을 개선했습니다.
    *   수강 이력이 없는 사용자의 경우, 장바구니 항목이 '1학기'에 잘못 추가되던 버그를 수정했습니다.
    *   장바구니의 마지막 항목 삭제 시, 수강 이력 화면이 즉시 업데이트되지 않던 문제를 해결했습니다.
