# use case 정리

\- 학생은 카카오로 \*\*회원가입/로그인\*\*을 할 수 있다."

\- "학생은 \*\*과목을 검색\*\*할 수 있다." (과목명, 교수명, 과목 코드로)

\- "학생은 \*\*전체 과목 목록\*\*을 볼 수 있다." (전공별/영역별 필터링)

\- "학생은 특정 \*\*과목의 상세 정보\*\*를 볼 수 있다." (선수 과목, 시간대, 강의실 등)

\- "학생은 \*\*자신이 들은 과목\*\*을 '수강이력'에 추가할 수 있다."

\- "학생은 \*\*전공 기반으로 과목을 ai에게 추천\*\*받을 수 있다."

\- “학생은 과목을 장바구니에 추가할 수 있다.”

\- "같은 전공이였던 익명의 선배 시간표를 조회/복사할 수 있다.”

\- “엇학기로 수강한 선배임을 인식할 수 있다”
